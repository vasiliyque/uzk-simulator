# Известные проблемы и TODO

## Проблемы

- [ ] **Axis kink в realistic mode**: сигнал ступеньки не генерируется при чистом переломе оси (без смещения кромок). `rayIntersectStep` не находит ступеньку, т.к. поверхность непрерывна в x=0. Решено только для direct/reflected.
- [ ] **Step wall drawing при axis kink**: рисует вертикальные линии в x=0 (ctx.lineTo(toX(0), toY(0))), хотя реальной ступеньки нет — поверхность гладкая.
- [ ] **`drawDimensionLines`**: использует `probeX` из своей логики, а не обновлённую. Может не совпадать при повороте оси.
