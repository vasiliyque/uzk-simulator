# Проект: Симулятор УЗК сварных швов

**Расположение**: `F:\Simulator U\26.05.2026 — opencode\Refactor\`
**Версия**: Opencode SI (модульная версия, JS/CSS разделены)

## Суть

Веб-симулятор ультразвукового контроля (УЗК) сварных швов для обучения и демонстрации. Позволяет:
- Настраивать параметры контроля (толщина H, угол α, расстояние L, частота, gain)
- Моделировать геометрию шва (V/X, одност./двухст., валики)
- Добавлять дефекты (пора, шлак, трещина, несплавление) и видеть их на A-scan
- Учитывать дефекты сборки (смещение кромок, перелом оси)
- Отображать траекторию луча, отражения, стробы, уровни чувствительности

## Архитектура

```
Refactor/
├── index.html          — страница + inline-стили (standalone-версия в Opencode SI.html)
├── css/style.css       — CSS-переменные, темы (тёмная/светлая)
└── js/
    ├── constants.js    — константы: материалы, ПЭП, частоты, формы валиков
    ├── physics.js      — физика: траектория луча, отражения, затухание, тени, дефекты
    ├── renderer.js     — отрисовка canvas: сечение шва (uzCanvas) + A-scan (displayCanvas)
    └── ui.js           — состояние (state), слайдеры, селекты, drag&drop, ресайзеры
```

## Стек

- Canvas 2D для отрисовки
- Никаких зависимостей
